## 🎧 Features

- Instant playlist sorting by artist then album  
- Responsive design for all screen sizes

## ⚡ Quick Start

```bash
# Clone this repo
git clone https://github.com/23ovii/spotify-sorter

# Enter the project folder
cd spotify-sorter

# Open index.html in your browser
open index.html

# Or serve with local server for better experience
npx serve
````

> 💡 No complicated setup. Just clone, configure your Spotify Client ID, and go!
